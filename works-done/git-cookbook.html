How to initialize a new repo

  1. cd into wanted directory
  2. run 
    git init

How to clone a repo from GitHub 

  1. git clone https://github.com/YOUR-USERNAME/YOUR-REPOSITORY

How to create a branch AND check it out 

  1. git checkout -b branch_name

How to resolve a merge conflict

  1. Attempt to merge
  2. Use
    git status
    to see all conflicts
  3. Open each conflicting file in your text editor
  4. Manually merge the code
  5. delete all added brackets
  6. Use 
    git add
    on all those files
  7. Commit what is in the staging area
